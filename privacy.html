Политика конфиденциальности

Эта страница объясняет, какие данные собираются через формы сайта «Шерстяной лоск» и как они используются.

1. Какие данные собираются

Через формы сайта могут передаваться только данные, которые пользователь указывает сам:

  • имя или удобное обращение;
  • контакт для ответа: телефон, email, Telegram или другой мессенджер;
  • описание задачи, выбранная услуга, комментарий или параметры заявки.

2. Для чего используются данные

Данные используются для записи на груминг и связи по выбранной услуге. Мы не используем данные для массовой рекламной рассылки без отдельного согласия.

3. Передача третьим лицам

Данные могут передаваться техническим сервисам, которые нужны для работы сайта и доставки заявки: хостинг, почтовый сервис, мессенджер или CRM. Такие сервисы используются только в объеме, необходимом для обработки обращения.

4. Хранение и удаление

Заявки хранятся столько, сколько нужно для обработки обращения и дальнейшей коммуникации по нему. Пользователь может попросить удалить или уточнить свои данные, написав на email ниже.

5. Контакт по вопросам данных

По вопросам обработки персональных данных можно написать на bul82@yandex.ru.

Отправляя форму на сайте, пользователь подтверждает, что указал данные добровольно и согласен на их использование для ответа по заявке.
